# git-keeper [![npm-check](http://img.shields.io/npm/dm/git-keeper.svg)](https://www.npmjs.org/package/git-keeper)
> Check for merged or outdated local or remote git branches

## Install

```bash
$ npm i -g git-keeper
```

## Requirements

* node >= 4
* git >= 2.0.0 (tested on git 2.6.3 on mac)
